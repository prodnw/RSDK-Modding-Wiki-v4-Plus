# PauseSfx

## Description
Pauses a specific sound effect, it'll have to be playing for it to take action.

## Parameters
`sfx`

:   The slot of the sound effect. It's recommended to the `SfxName` alias for this.

## Return Value
None.

## Syntax
```
PauseSfx(int sfx)
```

## Example
```
PauseSfx(SfxName[Test])
```
