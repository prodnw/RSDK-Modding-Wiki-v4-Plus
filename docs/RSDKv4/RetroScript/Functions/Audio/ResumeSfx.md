# ResumeSfx

## Description
Resumes a specific sound effect, it'll have to be already paused for it to take action.

## Parameters
`sfx`

:   The slot of the sound effect. It's recommended to the `SfxName` alias for this.

## Return Value
None.

## Syntax
```
ResumeSfx(int sfx)
```

## Example
```
ResumeSfx(SfxName[Test])
```
