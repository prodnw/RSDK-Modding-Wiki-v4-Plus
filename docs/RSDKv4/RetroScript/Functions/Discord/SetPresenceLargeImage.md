# SetPresenceLargeImage

## Description
Sets the larger image that will appear.

!!! note
    The image name you put in the function must be the same as the image name that you set in the art assets of your Discord application.

## Parameters
None.

## Return Value
None.

## Syntax
```
SetPresenceLargeImage("(your image name here)")
```

## Examples
```
SetPresenceLargeImage("greenhill")
```
